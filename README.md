# Kalimbabe

A kalimba tablature editor and player.

## Demo

Link will go here

```bash
# create a new project in the current directory
npm create svelte@latest

# create a new project in my-app
npm create svelte@latest my-app
```

## Running

```bash
yarn

yarn dev
```

## Building

To create a production version:

```bash
yarn
yarn build
```
